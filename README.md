# Don't Look Back 3D

A horror maze runner game in 3D. 

## Features
- **3D Graphics**: Built using Three.js.
- **Difficulity Levels**: Easy, Medium, Hard, and the final "Extreme Survival Mode".
- **Dynamic Audio**: Procedural and layered horror soundtrack.
- **Responsive Controls**: Desktop (WASD) and Mobile touch controls.

## How to Play
1. Open `game.html` in any modern web browser.
2. Select your difficulty.
3. Find the green exit zone in normal modes or survive the timer in Extreme Mode.
4. **Don't Look Back.**

## Deployment
This game is a static web app. You can host it on:
- **GitHub Pages**: Go to Settings -> Pages and enable it for the `main` branch.
- **Vercel / Netlify**: Drag and drop the folder to the dashboard.
